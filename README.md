# BILI PLUS

## 功能介绍

本项目是用于在 PC 浏览器使用 B 站时，作为谷歌浏览器插件，强化各种功能特性或提供额外的功能。
目前主要是提供字幕额外功能，可自动提取当前 B 站视频字幕并下载为为含时间戳的`.srt`字幕文件和用于 AI 笔记用的纯文本`.md`文件。

## 使用方法

1. 打开 Chrome → `chrome://extensions/`
2. 启用“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本目录
5. 打开任意 B 站视频详情页
6. 点击插件图标 → 找到按钮“提取字幕”

即可下载对应的字幕文件到当前设备。

## 功能

-   自动读取登录 Cookie（无需手动输入）
-   自动识别可用字幕语言
-   一键导出 `.srt` 文件
