# Bilibili 字幕提取器

一个 Chrome 插件，可自动提取当前 B 站 视频字幕并下载为 `.srt` 文件。

## 使用方法

1. 打开 Chrome → `chrome://extensions/`
2. 启用“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本目录
5. 打开任意 B 站 视频页
6. 点击插件图标 → “提取当前视频字幕”

会自动下载 `.srt` 文件。

## 功能

-   自动读取登录 Cookie（无需手动输入）
-   自动识别可用字幕语言
-   一键导出 `.srt` 文件

## 文件说明

-   `background.js`: 处理 Cookie 和字幕接口请求
-   `content.js`: 注入页面并下载字幕
-   `popup.*`: 控制界面
